# Number Guessing Game

A simple number guessing game built using **HTML**, **CSS**, and **JavaScript**.

The user selects a number range and has **7 chances** to guess the correct number. After each guess, the game gives feedback telling the user whether their guess is too high or too low. The game also keeps a record of previous guesses so users can avoid guessing the same number again.

## Features

- Three difficulty ranges:
  - 1 to 10
  - 1 to 100
  - 1 to 1000

- Users get 7 attempts to guess the correct number
- Feedback after each guess:
  - Too high
  - Too low
  - Correct guess

- Shows previous guesses made by the user
- Prevents users from easily repeating the same number
- Simple and clean user interface
- Built with only HTML, CSS, and JavaScript

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run the Project

1. Download or clone the project files.

2. Open the project folder.

3. Open the `NumGuess.html` file in your web browser.

No installation or extra dependencies are required.

## How the Game Works

1. The user chooses a range: 10, 100, or 1000.
2. The game randomly generates a number within the selected range.
3. The user enters a guess.
4. The game checks the guess and gives feedback:
   - If the guess is higher than the correct number, it tells the user the guess is too high.
   - If the guess is lower than the correct number, it tells the user the guess is too low.
   - If the guess is correct, the user wins.

5. The game stores and displays all previous guesses.
6. The user has a maximum of 7 chances to guess correctly.
